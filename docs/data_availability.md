# Data availability and privacy

The raw pelvic MRI scans are not distributed in this repository because clinical imaging data may contain protected health information in DICOM metadata, private scanner tags, acquisition dates, institution identifiers, and image-associated annotations.

The repository contains executable code, graph-output formats, generated non-identifiable tensors, metric tables, figures, and documentation needed to reproduce the public computational pipeline. Access to clinical MRI data requires institutional approval, ethics clearance, and a controlled data-use agreement.

The public execution artifacts are not clinical records and are not intended for diagnosis, treatment planning, or direct patient care.
